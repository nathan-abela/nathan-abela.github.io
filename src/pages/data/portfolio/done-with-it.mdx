Done With It is a React Native application developed with Expo CLI. It simplifies the selling process for unwanted items, offering a smooth user experience across its 10 screens. The app consists of features such as user authentication, form validation, and push notifications. Streamlining functionality and user convenience, Done With It redefines the process of parting with belongings.
